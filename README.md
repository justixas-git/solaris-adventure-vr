# Solaris Adventure — Educational VR Game

**Type:** Bachelor's Thesis Project
**Engine:** Unity
**Platform:** VR
**Focus:** VR interaction, gameplay systems, educational design

## Overview
Solaris Adventure is an educational VR game developed as part of my Bachelor’s thesis.  
The project aims to teach middle-school astronomy concepts through interactive VR missions and free exploration.

The game is structured around multiple themed missions and a free-roam mode, allowing players to explore concepts at their own pace.

## My Role
I was responsible for the full development of the project, including:
- VR interaction and control systems
- Gameplay logic and mission flow
- Scene structure and navigation
- UI flow and player feedback

## Key Features
- Modular mission-based structure
- Interactive VR learning mechanics
- Free=roam exploration mode
- Educational content integrated into gameplay

## Technology
- Unity (VR)
- C#
- XR interaction systems
- Custom gameplay scripts
